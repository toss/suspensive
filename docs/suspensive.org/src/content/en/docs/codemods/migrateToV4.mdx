# Migrate to v4

<Callout type='info'>

- Recommended when upgrading `@suspensive/react-query` from v3 to v4 — runs every code-level migration in a single command.
- Updating your `package.json` dependencies is still a manual step.

</Callout>

`@suspensive/react-query` v4 contains two code-level breaking changes:

1. The unified `@suspensive/react-query` package is removed. Imports must point at the version-specific `@suspensive/react-query-4` or `@suspensive/react-query-5`.
2. APIs backported into `@tanstack/react-query` (`useSuspenseQuery`, `queryOptions`, `mutationOptions`, `usePrefetchInfiniteQuery`, and so on) are removed from `@suspensive/react-query-{4,5}`. They must be imported directly from `@tanstack/react-query`.

Internally chains [`migrate-suspensive-react-query-package`](./migrateSuspensiveReactQueryPackage) followed by [`tanstack-query-import`](./tanstackQueryImport).

```bash filename="Terminal"
npx @suspensive/codemods migrate-to-v4 .
```

For example:

```tsx /@suspensive/react-query/
import {
  useSuspenseQuery,
  SuspenseQuery,
  queryOptions,
  IsFetching,
} from '@suspensive/react-query'
import { mutationOptions } from '@suspensive/react-query-5'
```

Transforms into:

```tsx /@tanstack/react-query/ /@suspensive/react-query-5/
import { useSuspenseQuery, queryOptions } from '@tanstack/react-query'
import { SuspenseQuery, IsFetching } from '@suspensive/react-query-5'
import { mutationOptions } from '@tanstack/react-query'
```

## Version detection

The codemod walks up from the current working directory and inspects the nearest `package.json` to determine whether you depend on `@tanstack/react-query` v4 or v5. The detected major version selects the destination package (`@suspensive/react-query-4` vs `@suspensive/react-query-5`).

If detection fails, the codemod falls back to **v5** with a warning. To force a target explicitly, set the `SUSPENSIVE_RQ_TARGET` environment variable to `4` or `5`:

```bash filename="Terminal"
SUSPENSIVE_RQ_TARGET=4 npx @suspensive/codemods migrate-to-v4 .
```

## Caveats

- The codemod does **not** update your `package.json`. After running it, remove `@suspensive/react-query` from your dependencies and ensure `@suspensive/react-query-4` or `@suspensive/react-query-5` is installed.
- For the TanStack Query v4 → v5 `<QueryClientConsumer>` prop change, run [`migrate-query-client-consumer-props`](./migrateQueryClientConsumerProps) separately.
