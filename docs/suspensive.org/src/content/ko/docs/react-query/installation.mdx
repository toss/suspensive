# 설치하기

<Steps>

### 지원 브라우저 확인하기

Suspensive 라이브러리는 모던 브라우저를 위해 최적화되었습니다. 아래 브라우저에 호환되도록 설정되어 있습니다.

<br />

<Table>
  <tbody>
    {[
      { browser: 'Chrome', version: '>= 51' },
      { browser: 'Firefox', version: '>= 53' },
      { browser: 'Edge', version: '>= 18' },
      { browser: 'Safari', version: '>= 11' },
      { browser: 'iOS', version: '>= 11' },
      { browser: 'Opera', version: '>= 38' },
    ].map((item) => (
      <Table.Tr key={item.browser}>
        <Table.Td>{item.browser}</Table.Td>
        <Table.Td className="text-center">{item.version}</Table.Td>
      </Table.Tr>
    ))}
  </tbody>
</Table>

> 환경에 따라, polyfills 추가가 필요할 수도 있습니다. 더 오래된 브라우저를 지원하려면, 스스로 node_modules에서 라이브러리를 transpile할 필요가 있습니다.

### 의존성 추가

사용 중인 @tanstack/react-query 버전에 맞는 패키지를 설치하세요.

<Tabs items={['@tanstack/react-query v5', '@tanstack/react-query v4']}>
<Tabs.Tab>

```shell npm2yarn
npm install @suspensive/react-query@npm:@suspensive/react-query-5 @tanstack/react-query@5
```

</Tabs.Tab>
<Tabs.Tab>

@tanstack/react-query v4는 @tanstack/react-query v5 보다 [더 낮은 버전의 브라우저를 지원](/ko/docs/react-query/motivation#v5는-구형-브라우저를-지원하지-않습니다)합니다.

```shell npm2yarn
npm install @suspensive/react-query@npm:@suspensive/react-query-4 @tanstack/react-query@4
```

</Tabs.Tab>
</Tabs>

<Callout type='info'>

npm alias를 사용하면 코드베이스 전체에서 import 경로를 `@suspensive/react-query`로 유지할 수 있습니다. @tanstack/react-query 버전을 변경할 때 package.json의 alias만 수정하면 되며, 코드 변경이 필요하지 않습니다.

</Callout>
